# Production Lane Boundary

## Current Correct Status

This repository is an A-grade public-safe executable review build and production-candidate track.

It is not a customer-managed production runtime.

## Public-Safe Review Build Means

- reviewer can run it
- proof cases execute
- receipt and replay behavior can be checked
- tampering can be detected
- buyer can see the seam
- MACI cannot bind
- workflow cannot bypass
- Elyria/VERITA remains protected underneath

## Production-Candidate Track Means

The repository contains visible templates, contracts, and gates for:

- auth/RBAC
- tenant isolation
- production signing requirements
- external verifier requirements
- tamper-evident audit chain
- threat model
- deployment hardening
- operational runbooks
- certification-track evidence

## Not Yet Claimed

- production certified
- commercial deployment ready
- customer-managed production runtime
- protected runtime dump
- full private Elyria kernel exposure
- regulatory authority
- medical/legal/financial authority

## Required Before Production Claim

Production deployment requires:

- private Elyria/VERITA integration
- production key management
- tenant isolation
- auth/RBAC
- deployment hardening
- monitoring and incident response
- written commercial scope
- customer security approval or external audit
