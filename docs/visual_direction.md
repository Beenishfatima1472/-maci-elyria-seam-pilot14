# MACI Visual Direction

## Design language

Not dark cyber.

Not crypto.

Not generic AI.

The visual surface should feel:

- premium
- calm
- transparent
- trustworthy
- governance-oriented
- executive-review ready

## Palette

Ocean Slate      #2E5266
Forest Green     #3D6B52
Sand             #D8C7A1
Cloud White      #F7F5EF
Copper           #B7793E
Soft Iris        #B8A7E8

## Holographic treatment

Use subtle iridescent borders and accent lines.

Do not use neon.

Do not use hard black backgrounds.

Use translucent panels with visible background depth.

## Executive surfaces

Hero banner
Executive dashboard
System map
Decision surface
Proof matrix

All visuals should support readability before decoration.
